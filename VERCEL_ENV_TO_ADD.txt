## HƯỚNG DẪN THÊM VÀO VERCEL:

### Cách 1: Thêm từng biến

1. Trong màn hình Environment Variables Vercel
2. Thêm 5 cặp Key-Value như trên
3. Click "Save"

### Cách 2: Import nhanh (Khuyên dùng)

1. Tạo file `vercel-env.txt` với nội dung bên trên
2. Click button "Import .env" trên Vercel
3. Select file
4. Click "Save"

---

⚠️ QUAN TRỌNG:
- Sau khi Save → Vercel sẽ tự động redeploy
- Đợi 2-3 phút để rebuild
- Website sẽ hoạt động với Cloudinary integration
